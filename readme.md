# Contributing

Contributions are more than welcome!

1. Fork the repo.
2. Make a branch.
3. Push your changes.
4. Open a PR!

To check your local changes as you make them, [load the unpacked extension in Chrome](https://developer.chrome.com/extensions/getstarted#unpacked).
